Bước 2: Khởi động Frontend (React + Vite)
Bạn hãy mở thêm một Terminal mới (bấm dấu + ở góc phải bảng Terminal) và thực hiện:

Gõ lệnh để vào thư mục frontend:
cd frontend
npm install
Chạy lệnh khởi động:

npm run dev
